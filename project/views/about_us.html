About Us


This application will exist to help website users in the selection process of a pet name. By combining multiple lists of pre-existing pet names, the user will be able to answer questions to a short quiz to help the random generator deploy more accurate name suggestions. The application will handle the parsing out of these lists depending on user input to quiz questions. These questions will range from things like pet gender to pet behavior. The final result of the application should be a small list of 1-5 unique names based on one's questions in the quiz. Therefore this application will provide some resolve to users who are indecisive when it comes to naming their pets. This application can also be used for more than naming real-life pets, it can also be used in creative writing projects or for virtual companions in RPG-like settings.
